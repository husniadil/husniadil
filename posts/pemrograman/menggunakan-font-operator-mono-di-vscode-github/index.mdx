Memakai custom _font_ membuat pengalaman _programming_-mu lebih menyenangkan, sekalinya pake gak mau balik lagi ke font lama.

Tapi harganya mahal, ngapain buang-buang duit segitu? Setelah _googling_ dengan keyword `operator mono`, di halaman pertama muncul website bernama cufonfonts buat download _font_ gratis, saya gak tahu apakah ini "bajakan" atau gak, kalau misal bajakan, kenapa gak di-_takedown_ font Operator Mono dari website itu. **Coba aja dulu**.

Setelah di-_download_, _install_ aja di komputer. Setelah ke-_install_, gimana nih cara pakainya di VSCode?

## Cara pasang Operator Mono di VSCode

Tinggal masuk ke menu `Preferences` > `Settings` aja. Trus cari _setting_ dengan _keyword_ `font`.

![VSCode Font settings](./halaman-pengaturan-font.png 'Halaman pengaturan font')

Nah dari situ klik aja `Edit in settings.json` (link di bawah `Font Ligatures`), maka akan muncul _editor json_, dan cari baris berikut.

```json
  // Controls the font family.
  "editor.fontFamily": "Consolas, 'Courier New', monospace",

  // Configures font ligatures.
  "editor.fontLigatures": false,
```

Ubah _font_-nya dengan menambahkan `'Operator Mono'` di depan `Consolas`-nya, dan mengubah setting `fontLigatures` menjadi `true`.

```json
  // Controls the font family.
  "editor.fontFamily": "'Operator Mono', Consolas, 'Courier New', monospace",

  // Configures font ligatures.
  "editor.fontLigatures": true,
```

Selain itu kamu harus _download_ _theme_ [Dark+ Mono](https://marketplace.visualstudio.com/items?itemName=cmckni3.dark-plus-mono) untuk mengubah _style_-nya.

Selesai _download_, kamu harus mengaktifkannya, masuk lagi ke `Preferences` > `Settings`, kemudian cari _setting_ dengan _keyword_ `workbench color theme`.

Ubah `Color Theme` menjadi `Dark+ Mono`. Kemudian buka salah satu _file_ dalam _project_-mu.

Nah beginilah tampilannya setelah kamu berhasil mengubah pengaturan _font_ dan _theme_.

![Example Operator Mono VSCode](./example-operator-mono-vscode.png 'Tampilan editor setelah menggunakan Operator Mono dan Dark+ Mono.')

Eits, gak cuma sampe di sini nih. Kamu juga bisa mengubah _default_ _font_ GitHub kamu.

## Cara pasang Operator Mono di GitHub

![Go by Example: Methods](./gobyexample-methods.png 'Tampilan GitHub dengan font Operator Mono.')

Gimana caranya tuh?

Asumsinya kamu pake _Chrome browser_, kamu tinggal pasang aja extension [Stylebot dari Chrome Web Store](https://chrome.google.com/webstore/detail/stylebot/oiaejidbmkiecgbjeifoejpgmdaleoha?hl=id). Trus masuk ke pengaturannya.

![Pengaturan stylebot](./pengaturan-stylebot.png 'Pengaturan **Stylebot**.')

Tambahkan style baru, klik `Add a new style...` kemudian isi formulirnya:

Di bagian `**URL**` masukkan `***_github.com_***`

Kemudian di bagian _textarea_-nya, isikan:

```css
.highlight .pl-c {
  font-style: italic;
}

.markdown-body .highlight pre,
table.highlight,
.blob-code-inner {
  font-family: 'Operator Mono';
  font-size: 14px;
  font-weight: 400;
  line-height: 1.5em;
}
```

Kemudian tekan tombol `Save`. Langsung saja buka salah satu _file_ _codingan_ di `GitHub`-mu untuk melihat hasilnya.
